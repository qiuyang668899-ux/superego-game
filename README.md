# 《超我》AI 赛博修仙

让未来的“超我”替你参悟万法、渡过诸劫，炼成神通后穿过镜门，把力量投回现实中的你。

> 愿力高于业力，能力才得以显化。学习是炼法的手段，修行与自我得救才是主线。

## 手机在线体验

[打开 GitHub Pages 版本](https://qiuyang668899-ux.github.io/superego-game/)

GitHub Pages 是纯前端预览：没有绑定安全服务端时，“镜门问道”会由本地心诀代守。要启用通用智能回答，请使用下方本地启动方式，或把仓库部署到支持服务端函数的平台并配置 `DEEPSEEK_API_KEY`。

## 本地预览

```bash
npm install
export DEEPSEEK_API_KEY="你的服务端密钥"
npm run start:local
```

浏览器访问 `http://127.0.0.1:4175/`。

密钥只存在服务端环境变量中，不会进入网页代码。未配置密钥时，应用仍可运行，但“镜门问道”会自动退回本地心诀。

## 智能心核

- DeepSeek V4 Flash 负责通用理解、事实回答、决策分析和教练式追问。
- 每次对话会携带最近 16 轮消息、用户主动保留的记忆、当前愿力、境界、已悟神通与少量相关修行资料。
- 回答不再固定套用“做两分钟”，会先判断用户是在问事实、做选择、表达情绪还是需要具体方法。
- 用户首次启用前必须明确同意；对话和长期记忆保存在本机，可随时清除。
- 云端失败或危机关键词触发时，应用会使用本地安全回应兜底。

## 核心循环

1. 缔结本命愿，唤醒来自未来的“超我”。
2. 把书籍、经典、观点、笔记或真实困境投入万法灵炉。
3. 超我从中取“道”、炼“法”、留“术”，形成一门可用神通。
4. 超我在彼岸代修渡劫，再通过镜门把神通投回现实。
5. 你只接住一个能完成的动作，让神通真正炼成自己的能力。
6. 生成可下载的修为战报，记录与传播每次突破。

## 内容与隐私

游戏进度、对话与长期记忆保存在浏览器 `localStorage`。只有用户明确开启“智能心核”后，当前问题、近期对话和经过截断的相关上下文才会发送到配置的 AI 服务。API 密钥由服务端代理保护，不会下发到浏览器。
