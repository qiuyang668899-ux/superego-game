## 发布后冒烟

- [ ] HTTPS 首页返回 200，核心静态资源无 4xx/5xx；HTTP 正文不直出并按预期跳转。
- [ ] `GET /api/health` 返回 200、`ok:true`、`production:true`，且不泄露密钥。
- [ ] `POST /api/coach` 用同源 Origin 返回结构化回答；异常 Origin 403，超长正文 413。
- [ ] HSTS、CSP（含 `frame-ancestors 'none'`）、X-Frame-Options、nosniff、Referrer-Policy 均存在。
- [ ] 390×844 与 360×640：序章到主循环、四导航、菜单滚动和关闭无回归。
- [ ] 旧存档迁移后进度不丢；导出生成 JSON；恢复点成功；损坏档显示恢复入口且不静默覆盖。
- [ ] 功德坊没有可用的体验购买，分享页没有模拟好友发奖入口。

## 发布后 5–10 分钟观察

- [ ] `superego.service` 无重启循环，Nginx 和应用 error 日志无新增致命错误。
- [ ] 首页/health/coach 的 2xx、4xx、5xx 比例合理；重点观察 403、413、429、503 是否异常突增。
- [ ] DeepSeek 调用延迟、上游错误率、并发拒绝和当日预算计数符合预期。
- [ ] CPU、内存、文件系统和共享预算文件写入正常；浏览器控制台无新增 error。
- [ ] 至少一台真实手机复验存档写入、刷新恢复和主线一步操作。

## 自动回滚触发条件

出现任一项立即停止观察并自动回滚到发布前 release：

- health 连续 2 次失败，或首页/coach 连续 5 次出现 5xx/无有效 JSON。
- `superego.service` 在 5 分钟内重启 2 次及以上，或 Nginx 配置/重载失败。
- 2 分钟窗口内应用 5xx 超过 2%，或 AI 预算/并发保护失效导致无界调用。
- 手机核心流程白屏、底部导航不可用、旧档迁移丢失或损坏档被新档静默覆盖。
- 必需安全头缺失、HTTP 暴露游戏正文，或异常 Origin 能成功调用 coach。

回滚后重新验证上一版本首页与 health；保留失败 release 和日志用于复盘，不修改真实玩家本地存档。
